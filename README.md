- Firebase

- React-Router-Dom.
- Netlify

# Getting Started:
# Visit 
 https://df-congo-marketplace.netlify.app/


1. Install the dependencies needed to run Congo by running in terminal:

        `npm install`

2. Make sure the Rails server is already running, in seperate terminal run:
 
        `npm start`
